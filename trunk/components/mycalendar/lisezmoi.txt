-------------------------------------------------------------------------------------
 DESCRIPTION
-------------------------------------------------------------------------------------
        MyCalendar est un script qui permet l'aide au remplissage des dates. En effet
	grâce à l'affichage d'une fenêtre calendrier, vous pouvez facilement trouver
	une date et ainsi la valider : celle-ci s'affichera automatiquement dans le
	formulaire appelant sous le format MySQL (YYYY-MM-DD)

-------------------------------------------------------------------------------------
 CONFIGURATION REQUISE
-------------------------------------------------------------------------------------
	_ Serveur d'application supportant le PHP (testé sur 4.05)
	_ Compatible Netscape 4+ et IE 4+

-------------------------------------------------------------------------------------
 INSTALLATION
-------------------------------------------------------------------------------------
	1. Copiez le fichier mycalendar.php dans le repertoire voulu de votre site.

	2. Essayez et éditez l'exemple pour apprendre à utiliser le script

	3. Si vous voulez initialiser le calendrier à une date precise ajoutez les
	   paramètres suivants lors de l'appel du script :
	   "jour=10&mois=9&annee=2001"

	4. Modifiez les couleurs, le texte en éditant le fichier mycalendar.php
